# MealFlow

Multi-user meal planning, inventory, and nutrition app.

## Setup

1. **Create a Google Sheet** with tabs for recipes, inventory, planning, and nutrition tracking
2. **Get a Google API Key** to access your sheet programmatically
3. **Deploy to GitHub Pages** to make the app accessible online

## Structure

- `js/` — JavaScript modules for sheet integration and data management
- `css/` — Styling for the web interface
- `index.html` — Main entry point
- `docs/` — Documentation including API setup guides

## Development

Run tests with Jest:
```bash
npm test
```

Start a local dev server:
```bash
npm run dev
```

## Deployment

Push to GitHub Pages to deploy the app online.
